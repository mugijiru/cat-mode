# cat-mode.el #

cat-mode.el は
[The Cat API](http://thecatapi.com "The Cat API")
を利用して猫の画像を表示するプラグインです。

## インストール ##

cat-mode.el をload-pathに置いて require してください。

## 使い方 ##

M-x cat-mode することで猫画像が表示されます。
cat-mode中は Enterキーを押すことで別の猫画像が
qを押すことでmodeを終了することができます。
